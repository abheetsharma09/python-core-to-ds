# 🛠 Project 1: Log File Analyzer

### 📝 Description
This script automates the process of auditing server logs. It scans a text-based log file, identifies lines containing specific error status codes, and generates a summary report.

### 🚀 Concepts Learned
* **File I/O:** Using `with open()` to safely handle reading and writing.
* **String Parsing:** Using the `in` keyword to find patterns in text.
* **Accumulators:** Using a counter variable within a loop.

### 📂 Files
* `log_analyzer.py`: The main Python script.
* `server_logs.txt`: Input file containing raw log data.
* `log_result.txt`: Output file generated by the script.
