# 📊 Project 03: CSV Student Grade Filter

### 📝 Description
A data processing tool that reads student records from a CSV file and categorizes them into 'Passed' or 'Failed' status based on their marks.

### 🚀 Key Technical Features
* **Append vs. Write Logic:** Learned the critical difference between `"w"` (overwrite) and `"a"` (append) modes to prevent data loss.
* **CSV Parsing:** Utilized the `csv` module to handle tabular data rows as lists.
* **Type Casting:** Converted string inputs to integers for logical comparisons.
* **Context Manager Usage:** Efficiently managed file resources using `with open()` to ensure files are closed automatically.

### 🛠 Skills Demonstrated
* **Logic Branching:** Using `if/else` for data categorization.
* **Data Cleansing:** Basic preparation of raw text data for analysis.
* **Multi-file I/O:** Handling input and multiple output streams simultaneously.
