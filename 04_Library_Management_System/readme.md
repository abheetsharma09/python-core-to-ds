# 📚 Python Library Management System (OOP)

A lightweight Library Management System built with **Python** and **JSON** to practice Object-Oriented Programming (OOP) principles. 

## 🚀 Features
- **Add Books:** Register new books with Name, Author, and ISBN.
- **Persistent Storage:** Saves data to a `BOOKS_DATA.json` file so data isn't lost when the program closes.
- **Search Functionality:** Search for books by Name or ISBN.
- **Issuing System:** Check book availability and update status to "Issued."

## 🛠️ OOP Concepts Applied
- **Classes & Objects:** Used to model real-world books and library operations.
- **Inheritance:** The `book_issue` class inherits from `book_add` to share data structures.
- **Encapsulation:** Logic is bundled within specific methods to keep the code organized.
- **File I/O:** Reading and writing structured JSON data.

## 📖 How to Use
1. Run the script: `python library_system.py`
2. Select `0` to add a new book to the database.
3. Select `1` to search for and issue a book.
