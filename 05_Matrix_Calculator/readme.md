# CLI-Based Matrix Calculator 🧮

A custom Python implementation of Matrix operations built from scratch using Object-Oriented Programming (OOP) principles. This project focuses on understanding how data structures are managed at a low level without relying on external libraries like NumPy.

## 🚀 Features
- **OOP Architecture:** Utilizes Python classes and dunder methods (`__add__`, `__sub__`, `__truediv__`) for intuitive operator overloading.
- **Dynamic Input:** Allows users to define custom dimensions and populate matrices via the command line.
- **Matrix Operations:**
  - Addition (`+`)
  - Subtraction (`-`)
  - Element-wise Division (`/`)
  - *Multiplication (Coming Soon... currently in development using nested logic)*

## 🧠 The Logic Behind the Project
This project was built to master the concept of **2D Lists** and **Nested Loops**. By avoiding high-level libraries, I implemented the "Postman Logic":
1. Traversing rows (`i` index).
2. Traversing columns (`j` index).
3. Performing element-wise calculations.
